# laptopGuideRobot
本项目是一个基于NIM开发的笔记本电脑导购智能机器人工具，用于帮助消费者快速找到符合自己心意的产品，该工具同时适用于商家和消费者个人，商家端上传商品信息，消费者根据需求寻求推荐建议。本项目主要使用Python语言进行编写，是NVIDIA第11届黑客松比赛的完成作品之一，开发团队为卡拉米，以下为我们项目的介绍视频地址：
